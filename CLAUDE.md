# BacklinkCAT Project

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (Icons)

## Core Guidelines

- Follow all rules in @AGENTS.md strictly.
- **SSR First**: All pages and components should be Server Components by default.
- **Separators**: Use `-` for all UI text separators.
- **Pixel Perfection**: 1920px width is the base for Figma screenshots.

## Folder Structure

- `src/app`: Routes and Page layouts.
- `src/components/sections`: Large page sections (SSR preferred).
- `src/components/ui`: Reusable atomic components.
- `public/images`: All image assets.
